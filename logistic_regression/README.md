# Logistic Regression From Scratch

Implemented Logistic Regression using only NumPy to understand how binary classification models learn without using machine learning libraries.

## Features

* Binary Classification
* Sigmoid Activation Function
* Binary Cross-Entropy Loss
* Gradient Descent
* Fully Vectorized NumPy Implementation

## Concepts Practiced

* Logistic Regression
* Probability Prediction
* Binary Classification
* Cross-Entropy Loss
* Gradients
* Parameter Optimization
* Matrix Operations
